# PianoCalligraphy

A personal catalog website for my mom's handwritten calligraphy pieces, each paired with a piano piece she plays alongside.

## About

My mom picked up calligraphy as a hobby in 2025. Watching her grow and find joy in it has been really meaningful to me, so I built this website as a way for her to easily look back on her work.

Each piece is a Bible verse, CCM song lyric, pastor's quote, or personal message — written by hand and photographed.

## Features

- **Search** — search across the reference, English text, and Korean text simultaneously
- **Filter** — filter by type (Bible Verses, Song Lyrics, Pastors & Speakers, Personal Messages)
- **Sort** — sort by date, Bible order, or alphabetically
- **Favorites** — heart pieces to save them to a favorites list
- **Language Toggle** — switch between English and Korean text
- **Request a Piece** — submit a request for a verse or lyric you'd like written

## Tech Stack

- Vanilla HTML, CSS, JavaScript
- Hosted on GitHub Pages
